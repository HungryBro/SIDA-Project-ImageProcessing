# Locked Original Multi-Test Control

โฟลเดอร์นี้เก็บชุดภาพควบคุมแบบหลายวัตถุจำนวน 30 ภาพจากชุดเดิม ซึ่งใช้สำหรับ
เปรียบเทียบโมเดล baseline เดิมกับ `baseline_v2` โดยเก็บไว้ภายใต้การทดลอง v2
เพื่อให้สามารถทำซ้ำการประเมิน CAM และ faithfulness ของ v2 ได้ในภายหลัง

ชุดภาพนี้ไม่ได้ถูกใช้สำหรับการฝึกโมเดลหรือการ validation
