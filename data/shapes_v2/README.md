# Synthetic Shapes Dataset v2

ชุดข้อมูลนี้ประกอบด้วยภาพสำหรับฝึกโมเดลทั้งแบบมีวัตถุเดี่ยวและหลายวัตถุ รวมถึงกรณีที่ยากขึ้น ภาพที่มีรูปทรงชนิดเดิมซ้ำกัน ภาพพื้นหลังเปล่า และชุดภาพหลายวัตถุแบบ locked-style สำหรับใช้เป็นชุดทดสอบควบคุม

โฟลเดอร์ `masks/` ใช้เก็บ mask ของวัตถุแต่ละชิ้น, `boundary_masks/` ใช้เก็บเส้นขอบของวัตถุ และ `rationale_masks/` ใช้เก็บบริเวณที่คาดว่าเป็นเหตุผลสำคัญต่อการจำแนก เช่น เส้นขอบของวงกลม หรือมุมของรูปหลายเหลี่ยม
