ภาพประกอบผลลัพธ์การคัดแยก (Concept Visualization)

การทำ Instance Segmentation ด้วย YOLO11-Seg:

จากตัวอย่างภาพจำลอง แสดงขอบเขตเส้นรอบเมล็ดข้าวหอมมะลิ 105 (Polygon Mask) สีสันต่างๆ ที่ถูกตรวจจับและวาดเส้นขอบแบบแยกรายเมล็ดได้อย่างหนาแน่นแม่นยำ

ขอบเขตพิกัดเส้นรอบตัววัตถุ (Mask Contour) ที่ได้จาก YOLO11-Seg นี้ จะถูกนำส่งเข้าสู่ไลบรารี OpenCV ในขั้นตอนถัดไปเพื่อวัดสัดส่วนทางเรขาคณิตหรือดึงสเปกตรัมสีเพื่อใช้ประเมินความสมบูรณ์ตำหนิท้องไข่/ข้าวด่างต่อไป

2 การตรวจจับข้าวหัก (Geometric Hybrid Approach)

แทนที่จะใช้ AI ในการเดาว่าข้าวเมล็ดไหนหัก (ซึ่งเกณฑ์จะไม่นิ่งและปรับแต่งยาก) เราใช้ คณิตศาสตร์เชิงเรขาคณิต (OpenCV) คำนวณจากรูปทรง (Mask) ที่ได้จาก YOLO11-Seg โดยตรง

ตัวอย่างการจำแนกข้าวหักตามรูปทรงเรขาคณิต (Crescent, Elliptical, Quadrilateral)

รายละเอียดทางเทคนิคของระบบคำนวณ

การคำนวณกรอบหมุนอิสระ (Minimum Area Bounding Box):
ใช้ฟังก์ชัน cv2.minAreaRect(contour) เพื่อสร้างสี่เหลี่ยมผืนผ้าแนบชิดกับเมล็ดข้าว โดยหมุนตามมุมเอียงจริงของเมล็ดข้าวนั้นๆ สกัดค่าด้านที่ยาวที่สุดเป็น ความยาว (Length) และด้านที่สั้นที่สุดเป็น ความกว้าง (Width) ทำให้วัดขนาดได้แม่นยำไม่ว่าเมล็ดข้าวจะหันไปทิศทางใด

ตัวอย่างการสร้าง Bounding Box แบบหมุนอิสระด้วย OpenCV

การคัดแยกด้วยสัดส่วนรูปร่าง (Aspect Ratio)

ตามมาตรฐาน ข้าวหอมมะลิเต็มเมล็ดจะมีสัดส่วนเรียวยาว อัตราส่วนความยาวต่อความกว้างเฉลี่ย >= 3.0

สูตรคำนวณอัตราส่วน (Aspect Ratio)

Aspect Ratio = Length / Width

โดยที่ Length = ด้านที่ยาวที่สุด, Width = ด้านที่สั้นที่สุด จาก minAreaRect

ข้าวเต็มเมล็ด (Perfect)
Aspect Ratio >= 2.5

เมล็ดเรียวยาว สมบูรณ์ ผ่านเกณฑ์คุณภาพ

ข้าวหัก (Broken)
Aspect Ratio < 2.5

เมล็ดสั้น แตกหัก ไม่ผ่านเกณฑ์คุณภาพ

หมายเหตุ: เกณฑ์ค่า 2.5 เป็นมาตรฐานเบื้องต้นที่สามารถปรับแต่งได้ตามความต้องการของอุตสาหกรรม โดยไม่ต้องเทรนโมเดลใหม่

เครื่องมือจำลองการคัดจำแนกข้าวหัก (Interactive Simulator)

เต็มเมล็ด
ข้อมูลวิเคราะห์จากกล้อง (Scanned Data)
ความยาวเมล็ดข้าว (Length): 120 px
ความกว้างเมล็ดข้าว (Width): 38 px
เกณฑ์ตัดสินข้าวหัก (Threshold) 2.50
